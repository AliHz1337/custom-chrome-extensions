About LeakDetect

LeakDetect is a browser extension designed to help users identify potential security vulnerabilities on websites they visit. It focuses on detecting exposed .git repositories and .env files, which can inadvertently reveal sensitive information.

Features

  • .git Repository Detection: Scans for publicly accessible .git directories that may expose source code and commit history.
  • .env File Detection: Looks for exposed .env files that may contain sensitive environment variables such as API keys, database credentials, and other configuration details.
  • User-Friendly Interface: Provides an intuitive popup interface for easy access to settings and detected vulnerabilities.
  • Customizable Settings: Allows users to enable or disable specific detection features based on their needs.

Usage

Once installed, LeakDetect runs in the background while you browse the web. If it detects any vulnerabilities on the current site, it will notify you through the extension's popup interface. You can then review the findings and take appropriate action.

Privacy

LeakDetect is committed to user privacy. The extension does not collect or store any personal data or browsing history. All detection processes are performed locally within your browser.
